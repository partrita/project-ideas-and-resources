# 프로젝트 아이디어 및 자료 ⚡💙⚡

<img src="https://user-images.githubusercontent.com/61475220/97093903-6d8aee80-166d-11eb-8799-13e119380d2b.jpg" width="50%" align=right>

무언가를 만들고 싶었지만 무엇을 해야 할지 전혀 몰랐던 적이 있나요? 작가들이 때때로 "작가 블록"을 겪는 것처럼 개발자들도 마찬가지입니다. 이 목록은 이 문제를 한 번에 해결하기 위해 만들어졌습니다! 👍

이 애플리케이션들은 다음과 같습니다:

  - 코딩 기술 향상에 좋습니다 :muscle:;
  - 새로운 기술을 실험하기에 좋습니다 🆕;
  - 다음 고용주/고객에게 깊은 인상을 주기 위해 포트폴리오에 추가하기에 좋습니다 :file_folder:;
  - 튜토리얼(기사 또는 비디오)에서 예시로 사용하기에 좋습니다 :page_with_curl:;
  - 완성하기 쉽고 새로운 기능으로 쉽게 확장할 수 있습니다 :ok_hand:;

이것은 단순히 프로젝트 목록이 아니라, 각 프로젝트를 처음부터 개발할 수 있을 만큼 충분히 자세히 설명하는 컬렉션입니다!

각 프로젝트에는 다음 **특징**이 있습니다:

1.  명확하고 설명적인 목표;
2.  구현해야 할 *사용자 스토리* 목록 (이 스토리들은 강제적인 *할 일* 목록이라기보다는 지침 역할을 합니다. 원한다면 자신만의 스토리를 자유롭게 추가하세요);
3.  기본 프로젝트를 개선할 뿐만 아니라 동시에 기술도 향상시키는 보너스 기능 목록 (이것들은 선택 사항이며 필수 사용자 스토리를 완료한 후에만 시도해야 합니다)
4.  프로젝트를 완료하는 데 필요한 것을 찾는 데 도움이 되는 모든 자료 및 링크
